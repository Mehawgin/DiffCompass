1、本示例需要用到两块显示模块，模拟显示一双眼睛；

2、将“RPi_Eye_Display”文件夹拷贝到Raspberry Pi目录下，然后运行如下命令：
   sudo chmod -R 755 RPi_Eye_Display/
   sudo ./rpi_eye.sh
   运行完成后，Raspberry Pi会自动重启，然后就可以看到眼睛显示了。

3、“Picture”目录下存放示例演示的图片和视频

4、接线说明如下：
   显示模块1接线：
   模块引脚                         Raspberry Pi引脚                        说明
     GND        ---->      GND（物理引脚6,9,14,20,25,30,34,39）           LCD电源地
     VCC        ---->         DC 5V/3.3V（物理引脚1,2,4,17）              LCD电源正极
     SCL        ---->                23（BCM：11）                        SPI0的时钟信号
     SDA        ---->                19（BCM：10）                        SPI0的写数据信号
     RES        ---->                31（BCM：6）                         LCD复位信号，低电平复位
     DC         ---->                29（BCM：5）                         LCD写数据和命令控制信号，高电平：数据，低电平：命令
     CS         ---->                24（BCM：8）                         LCD片选信号，低电平使能
     BLK        ---->                   不接                              LCD背光控制信号，低电平灭背光，不控制时，默认常亮   

   显示模块2接线：
   模块引脚                         Raspberry Pi引脚                        说明
     GND        ---->      GND（物理引脚6,9,14,20,25,30,34,39）           LCD电源地
     VCC        ---->         DC 5V/3.3V（物理引脚1,2,4,17）              LCD电源正极
     SCL        ---->                40（BCM：21）                        SPI1的时钟信号
     SDA        ---->                38（BCM：20）                        SPI1的写数据信号
     RES        ---->                31（BCM：6）                         LCD复位信号，低电平复位
     DC         ---->                29（BCM：5）                         LCD写数据和命令控制信号，高电平：数据，低电平：命令
     CS         ---->                36（BCM：16）                        LCD片选信号，低电平使能
     BLK        ---->                   不接                              LCD背光控制信号，低电平灭背光，不控制时，默认常亮   

以上两模块的RES和DC的接线引脚是公共的。
